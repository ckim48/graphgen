How to write formulas
Basics
  • Use variable x only (no other variables, no equal signs).
  • Operators: + - * / ^ (power), parentheses ( ).
  • Implicit multiplication is allowed: 2x, x(x+1), 3sin(x).
  • Constants: pi, E, oo.
